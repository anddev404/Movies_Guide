# Movies_Guide
Movies Guide app

Features of app
[x] Fetch details for movies playing now.
[x] Search movies based on keywords
[x] Support on different orientations
[x] Bookmark movies

Architecture used: MVVM

Components used to acheive:
[x] Coroutines
[x] Livedata
[x] Dagger2
[x] DataBinding
[x] Recyclerview
[x] Retrofit 2
[x] Gson
[x] Kotlin
[x] Proguard
